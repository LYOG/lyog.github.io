Java中的装箱和拆箱

Mar 15, 2018

一、什么是装箱？什么是拆箱？

Java为每种基本数据类型都提供了对应的包装器类型 ，基本类型到包装器类型之间的转换就是装箱和拆箱。

  • 装箱：自动将基本数据类型转换为包装器类型
  • 拆箱：自动将包装器类型转换为基本数据类型
基本数据类型 包装器类型
int（4字节） Integer
byte（1字节） Byte
short（2字节） Short
long（8字节） Long
float（4字节） Float
double（8字节） Double
char（2字节） Character
boolean（未定） Boolean

二、装箱和拆箱的实现

装箱过程是通过调用包装器的valueOf方法实现的，而拆箱过程是通过调用包装器的 xxxValue方法实现的。（xxx代表对应的基本数据类型）。
